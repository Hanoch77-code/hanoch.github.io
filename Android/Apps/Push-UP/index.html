Push-UP

This is app count number of push ups done by you.
This is done using luminance sensor in the phone
When the phone is under us while performing the push up, the value of the luminance changes.
According to my testing of luminance value, I found out that the luminance is less than 5.
We detect when the value goes lower than five and then expect it to go higher which determines if the value of the counter to be increased.
Additionally we added a start and reset button to begin and reset the value respectively.
Now, we are working on the development of the way to store this data for analysis and to note the progress.
